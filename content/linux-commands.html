Linux Commands Basics

Linux commands are powerful tools for interacting with your system, performing tasks efficiently, and automating processes. Below are some of the basic Linux commands that every user should know:

1. `pwd`

The `pwd` command stands for "print working directory". It displays the path of the current directory you are in. 

pwd

2. `ls`

The `ls` command is used to list the contents of a directory. By default, it shows the contents of the current directory. 

ls

3. `cd`

The `cd` command stands for "change directory". It is used to move between directories. 

cd /path/to/directory

4. `mkdir`

The `mkdir` command is used to create new directories. 

mkdir new_directory

5. `rm`

The `rm` command is used to remove files or directories. 

rm filename
rm -r directory_name

6. `cp`

The `cp` command is used to copy files or directories. 

cp source_file destination_file
cp -r source_directory destination_directory

7. `mv`

The `mv` command is used to move or rename files and directories. 

mv old_name new_name
mv file_name /path/to/destination

8. `chmod`

The `chmod` command is used to change the file permissions. 

chmod 755 filename

9. `chown`

The `chown` command is used to change the ownership of a file or directory. 

chown user:group filename

10. `man`

The `man` command is used to display the manual pages for other commands. 

man command_name

These commands form the foundation of working with Linux. Mastering them will enable you to navigate and manipulate your system with ease.
